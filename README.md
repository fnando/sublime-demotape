# Sublime Text — Demo Tape

This repo is just a mirror of the demo tape extension present at
https://github.com/fnando/demotape and exists to facilitate PackageControl
integration.

Any issues or pull requests should be directed to the original repo.

## Installation

### Setup Package Control Repository

1. Follow the instructions from https://sublime.fnando.com.
2. Open the command pallete, run “Package Control: Install Package“, then search
   for “Demo Tape“.

### Git Clone

Clone this repository into the Sublime Text “Packages” directory, which is
located where ever the “Preferences” -> “Browse Packages” option in sublime
takes you.
